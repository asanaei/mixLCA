# Eigenvalue Scree Plot for a Spectral Model

Eigenvalue Scree Plot for a Spectral Model

## Usage

``` r
.plot_spectral_scree(model, max_eigs = 20L)
```

## Arguments

- model:

  A `mixLCA` object.

- max_eigs:

  Maximum number of leading eigenvalues to display per class.

## Value

A `ggplot` object.
