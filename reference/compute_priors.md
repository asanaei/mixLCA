# Compute Prior Probabilities from the Concomitant Model

Compute Prior Probabilities from the Concomitant Model

## Usage

``` r
compute_priors(X, coefs)
```

## Arguments

- X:

  Numeric matrix (N x P), includes intercept.

- coefs:

  P x (K-1) coefficient matrix.

## Value

Numeric matrix (N x K) of prior class probabilities.
